How to run my code:
    1. put the json file into resources file 'src/main/resources'
    2. gradle build
    3. gradle run

Description of JSON format:
    The format of the position of cloud and step is in array, it is easier to manage  number and 
position. The image path for each entity is written after the position of the entity. The array for
enemy contains every information or data to construct a enemy. All numbers should be written
as float. There are some comments for some entities.
